# Bookmarker

一个可直接安装到 Tampermonkey 的本地书签保存脚本。

手动点击页面右下角按钮后，脚本会把当前页面的标题和链接写入你指定的本地 Markdown 文件中。现在脚本会使用一个固定的共用入口页来持有目标文件句柄，因此不同网站之间也能持续写入同一个文件。

## 功能

- 手动保存当前页面
- 使用共用入口页统一管理目标 `.md` 文件
- 后续跨网站持续追加到同一个 Markdown 文件
- 保存为 Markdown 表格
- 按链接自动查重，已存在则提醒且不写入
- 支持“保存页面（带备注）”，会弹出自定义备注面板并写进 `remark`
- 支持在入口页中更换目标文件、查看备份目录
- 支持快捷键 `Alt + Shift + B`

## 保存格式

```md
# Bookmarks

| No | 标题 | 链接 | remark |
| --- | --- | --- | --- |
| 1 | 页面标题 | <https://example.com/article> |  |
| 2 | 另一个页面标题 | <https://example.com/other> |  |
```

普通“保存当前页面”会先按链接查重；如果同一个链接已经存在，脚本会提示“该链接已存在，未更新文件”。

“保存页面（带备注）”的行为不同：

- 如果链接不存在，会新增一行并写入备注
- 如果链接已存在且 `remark` 为空，会更新该行的 `remark`
- 如果链接已存在且 `remark` 不为空，会先把原备注带入弹窗，修改后再保存

## 安装方法

1. 安装 Tampermonkey
2. 新建脚本
3. 复制 [`Bookmarker.user.js`](./Bookmarker.user.js) 全部内容
4. 保存并启用

## 使用方法

1. 打开任意网页
2. 点击右下角圆形 `B` 按钮展开面板
3. 点击 `保存当前页面`
4. 如果要写备注，点击 `保存页面（带备注）`，在自定义备注面板里输入或修改内容
5. 第一次使用时，脚本会打开共用入口页 `https://example.com/?bookmarker-hub=1`
6. 在入口页里点击 `更换目标文件`，选择一个已有的目标 `.md` 文件
7. 以后在任意网站继续点击同一个按钮，都会往同一个文件里的表格新增一行
8. 如果要换文件或查看备份，直接打开入口页操作即可

## 说明

- 推荐浏览器：Chrome / Edge 最新版
- 依赖浏览器的本地文件写入能力
- 共用入口页固定使用 `https://example.com/?bookmarker-hub=1` 作为统一 origin
- 目标 Markdown 文件请先在本地创建好，再用脚本选择
- 如果浏览器或 Tampermonkey 对文件权限隔离更严格，少数情况下可能需要到入口页重新选一次目标文件
- `更换目标文件` 和 `打开备份目录` 这类需要用户手势的操作，请在入口页中点击对应按钮完成
